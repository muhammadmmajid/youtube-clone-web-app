# YouTube-Inspired Web Application

## Overview
Developed a responsive multi-page web application inspired by a video streaming platform, focusing on layout design and user interface structure.

## Technologies
- HTML
- CSS (Flexbox, Grid)

## Features
- Responsive layout for different screen sizes  
- Structured navigation with header and sidebar  
- Reusable UI components (video grid, thumbnails)  
- Embedded video playback using iframe  

## Outcome
Built a clean and responsive front-end interface, improving understanding of layout design and modern CSS techniques.
